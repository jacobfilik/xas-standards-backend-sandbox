Run in a container
==================

Pre-built containers with xas-standards-backend-sandbox and its dependencies already
installed are available on `Github Container Registry
<https://ghcr.io/jacobfilik/xas-standards-backend-sandbox>`_.

Starting the container
----------------------

To pull the container from github container registry and run::

    $ docker run ghcr.io/jacobfilik/xas-standards-backend-sandbox:main --version

To get a released version, use a numbered release instead of ``main``.
